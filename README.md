# Agrinho-2026

# Agro Forte, Futuro Sustentável: equilíbrio entre produção e meio ambiente

## 🎯 Objetivo do Projeto
Este site foi desenvolvido como projeto para o Concurso Agrinho 2026. O objetivo principal é conscientizar estudantes e produtores rurais sobre a importância das práticas agrícolas sustentáveis, demonstrando que o crescimento da produção pode caminhar em perfeita harmonia com a preservação ambiental.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Layout responsivo (Flexbox/Media Queries) e gerenciamento de temas (Variáveis CSS).
- **JavaScript (ES6+):** Lógica interativa para saudação personalizada, simulador de economia de recursos e alternador de acessibilidade.
